# iris-analysis
# 🌸 Iris Dataset Analysis

## 📌 Project Overview

This project explores the **Iris dataset**, one of the most well-known datasets in data science and machine learning. The dataset contains measurements of iris flowers from **three different species**:

* **Setosa**
* **Versicolor**
* **Virginica**

Each species contains **50 samples**, giving a **total of 150 observations**.

The aim of this analysis is to examine the **key features that distinguish the species** and understand how petal and sepal measurements vary across the three flower types.

---

# 📂 Dataset Features

The dataset includes **four numerical features** used to analyze the flowers:

1. **Sepal Length**
2. **Sepal Width**
3. **Petal Length**
4. **Petal Width**

These features help determine the physical differences between the species.

---

# 📊 Species Analysis

## 🌼 Setosa

* Number of samples: **50**
* Average **Petal Length:** 1.46
* Average **Petal Width:** 0.24
* Average **Sepal Length:** 5.01
* Average **Sepal Width:** 3.42

🔎 **Insight:**
Setosa has the **smallest petals** among the three species and relatively wider sepals.

---

## 🌿 Versicolor

* Number of samples: **50**
* Average **Petal Length:** 4.26
* Average **Petal Width:** 1.33
* Average **Sepal Length:** 5.94
* Average **Sepal Width:** 2.77

🔎 **Insight:**
Versicolor shows **moderate petal and sepal sizes**, placing it between Setosa and Virginica.

---

## 🌺 Virginica

* Number of samples: **50**
* Average **Petal Length:** 5.55
* Average **Petal Width:** 2.03
* Average **Sepal Length:** 6.59
* Average **Sepal Width:** 2.97

🔎 **Insight:**
Virginica has the **largest petals and longest sepals**, making it the largest species in the dataset.

---

# 📈 Petal Size Category Analysis

The dataset also categorizes petals into **size groups** based on petal length.

| Petal Size Category | Percentage of Total Petal Length |
| ------------------- | -------------------------------- |
| Large               | **78.57%**                       |
| Small               | **12.98%**                       |
| Medium              | **8.44%**                        |

🔎 **Insight:**

* **Large petal sizes dominate the dataset**, accounting for over **78%** of total petal length.
* **Medium-sized petals are the least represented**, suggesting that most flowers tend toward either very small or large petal sizes.

---

# 🎯 Conclusion

The analysis shows clear **morphological differences between the three iris species**:

* **Setosa** has the **smallest petals** and wider sepals.
* **Versicolor** shows **moderate measurements**.
* **Virginica** has the **largest petals and longest sepals**.

Among all features, **petal length and petal width are the most distinguishing characteristics**, making them strong indicators for identifying iris species.
